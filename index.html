What you will learn in 1week.

For Photographers

  • Learn to edit photographs like a professional
  • You will learn everything you need to know about camera raw and processing images
  • Apply post production processes (sizing, cropping, retouching, masks, coloring & all enhancement) to create great photos.
  • You will learn how to work selectively with lights
  • Summarize the basics of output and presentation of photos.
  • Master Adobe Photoshop.

Lessons

  1. Advance Mask > Creating Simple Luminosity Mask, Creating More advanced Luminosity Masks, Creating Masks Using Color Range
  2. Image Mode > Calculations
  3. Understanding Channels
  4. Blending Option
  5. Color Grading > Curves, Gradient Map, Selective Color, Color Balance, LUT, 3D Lut Creator, Infinite Color Panel, NBP
  6. Color Correction > Color Cast, Lens aberration, Lights & Dark, Channel Mixer.
  7. Correction/Retouching: Noise Reduction Through Filters, Reducing Noise Using Channels, Noise Reduction Using The Raw Plug-In, The Amazing Clone Tool, Working With The Healing Brushes, Removing Unwanted Background Objects, Working With Facial Features, Whitening Teeth, Adding A Dash Of Color, Moving Objects, Handling The Dreaded Red Eye Problem.
  8. Sharpening Images Working With Smart Sharpen, The Unsharp Mask Filter, Sharpening With The High Pass Filter, Focusing Attention With Sharpen And Blur, Using The Shake Reduction Filter.
  9. Removing Distortion and correcting Perspective, Lens Correction Using Camera Raw, Auto Lens Correction, Manual Lens Correction, Working With Perspective Crop, Using Perspective Warp
  10. Automation > PDF Presentation, Batch, Image Processing, Script, Contact Sheet Manipulations > Double Exposure, Composite.
  11. Manipulations > Double Exposure, Composite.

For Graphics Designers

  1. Pixels vs Vectors (File Formats, Color Modes)
  2. Selections & Mask Skills Magic Wand Selection Tool: Create a Transparent image, Quick Selection Tool: Create a Transparent image, Masks: Clipping Mask, Masks: Layer Mask using the brush tool, Layer Mask using shapes, Channel Mask
  3. Typography
  4. Camera Raw and Simple Retouching
  5. Smart Objects and Filters
  6. Shape Tools
  7. Curves
  8. Masks
  9. Color Management
  10. Color Range
  11. Actions
  12. 3D Basics

This Graphic design training will open up the world of digital graphic design and help you expand your skillset. Learn from creative professionals who gladly share their tips, techniques and workflows most widely used in Graphic design and start on your learning path today.
